## 🚀 Características

* Operación de suma.
* Operación de resta.
* Operación de multiplicación.
* Uso de librerías personalizadas (.h).
* Interfaz sencilla mediante consola.
* Compatible con GCC.

---

## 🛠️ Requisitos Previos

Para compilar y ejecutar este programa se requiere:

* GCC (GNU Compiler Collection).
* Git.
* Una terminal de comandos (Git Bash, CMD o PowerShell).
* Visual Studio Code (opcional).

---

## 💻 Instalación y Compilación

Compilar el proyecto:

```bash
gcc main.c -o calculadora
```

Ejecutar el programa:

```bash
./calculadora
```

## 📚 Librerías Utilizadas

### Biblioteca estándar

```c
#include <stdio.h>
```

Permite utilizar funciones de entrada y salida como `printf()` y `scanf()`.

### Biblioteca personalizada

```c
#include "operaciones.h"
```

Contiene las funciones:

* suma()
* resta()
* multiplicacion()

Estas funciones realizan las operaciones matemáticas solicitadas por el usuario.
